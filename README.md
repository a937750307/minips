# 迷你PS

迷你PS 是一款基于 HTML5 技术的在线图像编辑器，无需购买、下载、安装或依赖过时的 Flash，也没有任何广告。核心功能包括：图层、滤镜，可作为开源的 Photoshop 替代品。

迷你PS 直接在浏览器中运行。你可以通过剪贴板粘贴（Ctrl+V）或从本地上传（通过菜单或拖拽）来创建图像。所有操作都在浏览器本地完成，不会上传到任何服务器。

## 功能特性

**文件**：打开图片、目录、URL、Data URL、拖拽上传、保存为 PNG/JPG/BMP/WEBP/动画 GIF/TIFF/JSON（图层数据）、打印。

**编辑**：撤销、剪切、复制、粘贴、选区、剪贴板粘贴。

**图像**：信息查看、EXIF、裁剪、缩放、调整大小（Hermite 重采样、默认调整）、旋转、翻转、色彩校正（亮度、对比度、色相、饱和度、明度）、自动色彩调整、网格、直方图、反色。

**图层**：多层系统、图层差异、合并、拼合、透明度支持。

**效果**：黑白、模糊（盒式、高斯、堆叠、缩放）、凸出/收缩、降噪、去饱和、抖动、点屏、边缘、浮雕、增强、伽马、颗粒、灰度、热力图、JPG 压缩、马赛克、油画、sepia、锐化、曝光、移轴、暗角、自然饱和度、复古、蓝图、夜视、铅笔，以及 Instagram 滤镜：1977、aden、clarendon、gingham、inkwell、lo-fi、toaster、valencia、x-pro ii。

**工具**：铅笔、画笔、魔棒、橡皮擦、填充、取色器、文字、裁剪、模糊、锐化、去饱和、克隆、边框、精灵、关键点、颜色缩放、替换颜色、恢复透明度、内容填充。

**帮助**：键盘快捷键、多语言翻译。
